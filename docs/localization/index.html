Localization

利用者の言語で結果を伝える

日本語・英語を標準搭載し、OS設定を自動検出します。独自言語も依存追加なしで登録できます。

言語を明示する

地域付きの ja-JP や en_US も基底言語として解釈します。取得不能または未対応なら英語へ戻ります。

python 
niltest.configure(language="ja")
niltest.configure(language="en")

アプリやプラグインから拡張

組み込みテンプレートと同じキーを翻訳し、起動時に登録します。キー不足や未知キーは登録時に検出されます。

python 
from niltest import register_locale

register_locale("fr", messages)

このサイトの翻訳方式

ページ構造、コード例、レイアウトは共有し、翻訳ファイルには文章キーだけを置いています。新しい言語ファイルを追加するとルートが自動生成されます。

i

翻訳されていないキーは日本語へフォールバックするため、Markdownページ全体を複製する必要がありません。
