Executable checks

仕様を本物の実装へ当てる

ケースのgivenで関数を実行し、実際の戻り値をreturnsと比較します。

Python API

python 
import niltest

niltest.configure(mode="test")  # @scenarioより前
from your_package.shipping import shipping_fee

result = niltest.run_tests(shipping_fee)
assert result.success
print(result.to_dict())

CLI

人が読む結果と機械可読JSONを選べます。モジュール内に登録されたシナリオだけを実行します。

shell 
niltest run your_package.specs
niltest run your_package.specs --json

モジュールを設計図として確認

inspectコマンドは関数シグネチャ、入出力型、ケース数、モック可能性、入力診断を一覧またはJSONで出力します。

shell 
niltest inspect your_package.services
niltest inspect your_package.services --json

終了コード

  • 0: すべての仕様ケースが成功
  • 1: ひとつ以上の仕様ケースが失敗
  • 2: 引数、インポート、対象モジュールに問題がある
